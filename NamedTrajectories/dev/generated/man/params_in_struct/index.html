    Parameters

    NamedTrajectory.jl support passing parameters as a Tuple when construct a NamedTrajectory.

    using NamedTrajectories

    First we need to define number of timesteps and timestep

    T = 10
dt = 0.1
    0.1

    then build named tuple of components and data matrices.

    components = (
    x = rand(3, T),
    u = rand(2, T),
    Δt = fill(dt, 1, T),
)
    (x = [0.848371087043323 0.4319989625941729 … 0.7768359517431934 0.1650579778977992; 0.49720030996286835 0.5805288349522133 … 0.7192814157700399 0.5068498432789058; 0.7667840274810969 0.33748700658022657 … 0.9564060429798049 0.13612487844854548], u = [0.3272539637175155 0.054903004062739846 … 0.8641841643329283 0.39903749553438617; 0.9448944279272858 0.21771217685006017 … 0.18691359276609432 0.9084840658288649], Δt = [0.1 0.1 … 0.1 0.1])

    we must specify a timestep and control variable for the trajectory.

    timestep = :Δt
control = :u
    :u

    some global params as a NamedTuple

    params = (
    α = rand(1),
    β = rand(1)
)
    (α = [0.15137792495392188], β = [0.5302110259816413])

    we can now create a NamedTrajectory object with parameters specification.

    traj = NamedTrajectory(components, params; timestep=timestep, controls=control)
    T = 10, (x = 1:3, u = 4:5, → Δt = 6:6), (α = 1:1, β = 2:2)
